¿Cómo funciona?

1. Crea una hoja en Google Sheets

2. En Extensiones → Apps Script, habilita la API

3. Pega la URL y el ID de tu hoja aquí

4. Cada cambio se sincroniza automáticamente

La hoja se crea con pestañas: Ingresos, Gastos Fijos, Gastos Variables, Proyección
